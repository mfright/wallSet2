■1.はじめに■
本ソフトは、好きな画像と共にカレンダーをデスクトップ背景に表示させるソフトです。





■2.起動方法■
使い方を以下の動画で紹介しています。
https://youtu.be/xl3q6y0YHFg


本ツールの起動には最新のJRE(Java Runtime)が必要です。
あらかじめダウンロードしてインストールしてください。
(https://java.com/ja/download/)


本ツールを起動するには、「2_OpenFileBackup_Windows.vbs」
をダブルクリックします。

２回目以降のバックアップでは、
「3_OpenFileBackup-RunBackup_Windows.bat」を実行することで、
設定画面を開けずに即バックアップ処理を開始させることができます。
このバッチファイルへのショートカットをスタートアップに登録させておけば、
PCへのログイン時に毎回自動的にバックアップさせることもできます。



【2-2.Linuxの場合】
本ツールの保存されたディレクトリで
「java -jar 1_OpenFileBackup.jar」コマンドを実行します。

２回目以降のバックアップでは、
「java -jar [ofb_core.jarのフルパス]」を実行することで、
設定画面を開けずに即バックアップ処理を開始させることができます。
cronでこのコマンドを実行させることで、
定期的かつ自動的にバックアップさせることもできます。




■3.つかいかた■
まず、「バックアップ元フォルダ」に、
バックアップしたいファイルが格納されたフォルダを指定します。

次に、「バックアップ先フォルダ」に、
ファイルのバックアップの保管先を指定します。

(ファイルサーバの例： \\192.168.0.10\backup )
(FTPサーバの例     ： ftp://yourhost.com/backup )
(外部ストレージの例： E:\backup )

最後に、「保存してバックアップ実行」を選択すると
バックアップが実行されます。

２回目以降のバックアップでは、
ファイルが追加・更新されたもののみコピーされます。
(ファイル更新時刻の差が2分未満のものを除く)




■4.オプション■
バックアップの設定は、「設定１」タブから「設定３」タブまでの
３つまで設定できます。

「このバックアップは実行しない(無効化)」にチェックが入った設定だけは
バックアップが実行されません。



２回目以降のバックアップ時に、バックアップ元で削除されたファイルは、
バックアップ先でも削除します。(FTP使用時を除く)

ただし、バックアップ先では削除せず退避させておきたい場合は、
「バックアップ元で削除されたファイルを、バックアップ先で削除せず、
以下フォルダへ保管する(FTP不可)にチェックを入れ、
削除されたファイルの退避先を指定してください。



バックアップ先に共用NASや共用FTPサーバを利用しているときは、
バックアップファイルを暗号化することをお勧めします。

暗号化する場合は、
「バックアップ先では以下のパスワードで暗号化する」
にチェックを入れ、パスワードを指定してください。



「バックアップ対象から除外するフォルダ」では、
バックアップ元フォルダに含まれているファイルのうち、
バックアップしないファイルやフォルダを指定します。
